My First Big Project

This game is a platformer where you play as a small flame that needs to make his way to the couch and set it on fire before burning out.

Project Link

The Process

This project was made in 60 hours for a gamejam in a team of 4. We had 2 C++ programmers who didn’t have much experience in unity and 2 unity developers including me, but the other unity dev was exclusively an artist so I had to both keep the project on track as well as help the other programmers to program a lot.

Challenges

My in game contributions: - All sounds
- Burning down mechanic
- Flame strength mechanic
- Death screen
- Level setup in unity using art made by artist
- Lighting
- Respawn point & candle
- Making webgl build
- Particles

What did I learn

In my role of teamlead or unofficially "Supreme Team Leader", I learned how to manage a small team and make sure we were able to make the tight deadline of the gamejam. I learned that to be a good lead you've gotta listen first and resolve the issue after.
